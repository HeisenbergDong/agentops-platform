# 14 Role Workspace Chat Rules

Each role workspace has:

- Left panel: capability summary, loaded rules, tools, input contract, output contract, prohibitions, model config, recent memory.
- Right panel: chat window for human capability additions, tests, rewrites, and questions.

Human additions can become:

1. Temporary instruction for current job or round.
2. Long-term role memory.
3. Rule change proposal handled by the Rule Collector.

Rule changes must show preview and require confirmation.
Role memory must record source message and timestamp.
